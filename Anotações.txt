IDE: Azure Data Studio e/ou MySQL workbench
Linguagem: SQL (aprender!)

DDL - Data Definition Language: (nivel de banco de dados)
-CREATE TABLE - Criar tabela
-ALTER TABLE - Altera os dados da tabela
-DROP TABLE - remove a tabela

DML - Data Manipulation Language: (nivel de tabela)
-INSERT - adição de dados na tabela
-UPDATE - alterações de dados na tabela (Mais importante)
-DELETE - deletar dados (registros dentro de uma tabela)

DQL - Data Query Language (Leitura ou consulta de dados em um banco)
-SELECT - quando utilizado sozinho é um comando de consulta
        - quando utilizado com alguma regra, torna-se um comando de manipulação de dados.

DCL - Data Control Language (Comumente atribuida ao DBA)
-GRANT - primeiro comando. Garante o acesso ao banco de dados
-REVOKE - permite um acesso restrito
-DENY - nega o acesso total a consulta de tabelas

PRINCIPAIS COMANDOS SQL:

- INNER JOIN                *COMPARA TABELAS QUE POSSUEM A MESMA INFORMAÇÃO
- CREATE / ALTER            *CRIA TABELAS E BANCO DE DADOS/ ALTERA OS BANCOS DE DADOS E TABELAS
- WHERE                     *IDENTIFICA O QUE QUEREMOS TRAZER DE RESPOSTA
- CHECK                     *CLAUSULA DE CONSULTA
- UPDATE                    *ALTERAR REGISTROS
- GROUP BY                  *AGRUPAMENTO DE REGISTROS
- HAVING                    *FUNCIONA COMO O WHERE, POREM É MAIS ESPECIFICO
- AVG()                     *AVERAGE MEDIA - CONTABILIZA OS DADOS COLETADOS
- AS                        *UTILIZA PARA ALTERAR TEMPORARIAENTE O NOME DE COLUNAS PARA FACILITAR A CONSULTA 
- ORDER BY                  *ORDENA OS DADOS COLETADOS NA CONSULTA DE FORMA ASCENDENTE OU DECRESCENTE
- COUNT                     *TRAZ A QUANTIDADE TOTAL DE REGISTROS
- DELETE                    *EXCLUI REGISTROS
- INSERT                    *INSERE REGISTROS
- LIKE                      *OTIMIZAR A CONSULTA
- WITH OWNERACCESS OPTION   *DA ACESSO DE ADMNISTRADOR DE FORMA TEMPORARIA
- GRANT
- REVOKE
- DENY
- DROP                      *REMOVER UMA TABELA
